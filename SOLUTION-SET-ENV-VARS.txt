Problem: Backend returns 502 Bad Gateway
Root Cause: App crashes on startup because database environment variables are missing

Your app requires these environment variables:
- DB_HOST
- DB_PORT
- DB_USER
- DB_PASSWORD
- DB_NAME

These are NOT configured in your Elastic Beanstalk environment!
